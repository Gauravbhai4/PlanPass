Khata rules

khata-rules.md
  • A-Khata — fully compliant. Required for building plan approval.
  • B-Khata — properties with violations. Cannot get plan approved. Convert to A-Khata first.
  • E-Khata — electronic form. Mandatory since 1 July 2025.
  • Conversion drive — 100-day B→A drive from 1 Nov 2025.

Nambike Nakshe

nambike-nakshe.md
  • Self-certification track for sites up to 50×80 ft
  • Residential only · A-Khata only
  • Licensed architect personally certifies the plan
  • Fast-track desk review
  • Architect liability — false certification = license suspension + ₹2L+ fine

Parking (2026)

parking-norms.md
  • 1 car per dwelling unit ≥ 1,000 sq ft
  • +10% visitor parking (multi-family)
  • 20% of spaces must have EV charging readiness (new in 2026)
  • Mechanized parking exempts ramp requirement
  • No parking in setback area

KTCP Act 1961

ktcp-act-1961.md
  • State foundational law for urban planning
  • §17 — individual plots need planning authority approval for A-Khata
  • §18 — use of building must match zone
  • §19 — authority approves or rejects plans
  • §76 — penalty for unauthorized development

About Permit Setu

Your bridge to BBMP / GBA building plan approval. Built on Azure OpenAI, FastAPI, and a skills-first architecture, grounded in the KTCP Act 1961 and the amended 2026 Zonal Regulations.

Tech stack

  • Backend — Python 3.10 · FastAPI 0.115 · Uvicorn
  • LLM — Azure OpenAI (gpt-4o / gpt-5.2)
  • Vision — PyMuPDF renders pages to JPEG · sent to vision API
  • Text PDFs — pypdf for corrections letters
  • Frontend — vanilla HTML / CSS / JS · Inter font
  • Config — python-dotenv · pydantic validation

Inspired by

CrossBeam by Michael Brown — winner of Anthropic's "Built with Opus 4.6" Hackathon (Feb 2026), targeting California ADU permits. Permit Setu ports the same architecture to BBMP/GBA approvals in Bengaluru, with Karnataka-specific skills and Azure OpenAI in place of Anthropic.

What's grounded

  • KTCP Act 1961 (§14B, §17, §18, §19, §24, §76)
  • Revised Master Plan (RMP) 2015 zonal designations
  • Amended Zonal Regulations effective 5 Jan 2026
  • BBMP Building Bye-Laws 2003 (amended)
  • Greater Bengaluru Governance Act 2024
  • National Building Code 2016 references

Status

Prototype. Built as a demonstration of how AI agents can be grounded in jurisdiction-specific regulations using a skills-first architecture.

Not a substitute for a licensed architect's professional judgment. Always verify outputs before submitting to BBMP/GBA.
