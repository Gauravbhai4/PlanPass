# Khata Rules — BBMP/GBA (2025-2026)

A Khata is a legal document linking a property to BBMP records for tax and approval purposes. **Without the right khata, no building plan can be approved.**

## Three Types of Khata

### A-Khata
- Issued for properties fully compliant with BBMP rules, taxation, and zoning
- **Required for building plan approval** under amended KTCP Act §17
- Can be used to apply for bank loans, building permits, water/electricity connections, registration

### B-Khata
- Issued for properties with violations (unauthorized layouts, missing approvals)
- **BUILDING PLAN APPROVAL IS NOT POSSIBLE WITH B-KHATA**
- Must convert to A-Khata first

### E-Khata (electronic)
- **MANDATORY since July 1, 2025** for all BBMP plan approvals
- Issued digitally via BBMP portal
- ~500,000 E-Khatas issued so far; BBMP processes ~3,000 per day

## B-Khata to A-Khata Conversion

A 100-day conversion drive began **November 1, 2025**, allowing B-Khata holders to convert to A-Khata.

### Conversion process
1. Apply via BBMP portal or designated centers
2. Submit ownership documents (sale deed, mother deed, tax receipts)
3. Pay applicable betterment charges and conversion fees
4. BBMP verifies and issues A-Khata

### Conversion fees (approximate)
- Betterment charge: 6% of guidance value
- A-Khata fee: 2% of stamp duty paid
- Plus penalty for unauthorized layouts (if applicable)

## ePID (Electronic Property ID)

- Unique identifier tied to E-Khata
- **Required for online plan submission** via bpas.bbmpgov.in
- Can be retrieved via Digilocker or BBMP portal

## Common Khata-Related Rejection Reasons

1. Submitting with B-Khata → automatic rejection
2. Missing latest property tax receipt → rejection
3. Mismatch between sale deed and BBMP records → flagged for verification
4. Property in an unauthorized/revenue layout → khata not issuable until layout regularized
5. Multiple owners not all listed on khata → consent letters required

## Pre-Submission Checklist for Khata

Before submitting any BBMP plan, verify:
- [ ] E-Khata in owner's name, current and active
- [ ] ePID retrievable from portal
- [ ] Property tax paid up to current quarter
- [ ] No pending dues or notices on the property
- [ ] Sale deed matches khata details (no typos, area matches)
- [ ] If multiple owners, NOC from each
